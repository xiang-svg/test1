# DriftScript 微信版 — 产品需求文档 V1.0

## 一、项目概述

### 1.1 产品定义
一款 AI 驱动的“电话传话”微信小游戏。用户输入一句话，经过 5-10 个不同 AI 人格的链式改写，最终展示语义漂移的全过程并给出“漂移分数”。

### 1.2 技术边界条件
- **部署平台**：GitHub Pages（纯静态托管）
- **架构限制**：无后端、无服务器、无数据库
- **AI 调用**：全部通过前端直接调用 **DeepSeek API**（`api.deepseek.com`）
- **API 密钥**：**硬编码于前端代码中，不介意泄露**
- **数据存储**：仅使用浏览器 localStorage
- **目标载体**：微信内嵌浏览器（H5），可封装为小程序 WebView

---

## 二、核心游戏流程

```
用户输入提示词
      ↓
选择 AI 人格链（默认随机 6 个）
      ↓
AI 人格 1 重写 → AI 人格 2 重写 → ... → AI 人格 N 重写
      ↓
每步实时展示重写结果 + 漂移动画
      ↓
计算最终漂移分数
      ↓
展示完整传递链条 + 一键分享
```

// 方案1：Jaccard 相似度（简单，前端友好）
function calcSimilarity(textA, textB) {
  const setA = new Set(textA.split(''));
  const setB = new Set(textB.split(''));
  const intersection = new Set([...setA].filter(x => setB.has(x)));
  const union = new Set([...setA, ...setB]);
  return intersection.size / union.size;
